# Weather Monitoring System

## Overview
This application continuously fetches weather data from the OpenWeatherMap API for major metros in India and processes it to provide real-time insights and alerts.

## Data Source
The system retrieves data from the OpenWeatherMap API, focusing on:
- Main weather condition (e.g., Rain, Snow, Clear)
- Current temperature in Celsius
- Feels-like temperature in Celsius
- Time of the data update (Unix timestamp)

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/aj240917/zeotap-assignment-rule-engine-weather-system.git
   cd zeotap-assignment-rule-engine-weather-system
   
2. Install dependencies:
pip install -r requirements.txt

3.Replace YOUR_API_KEY in weather_monitor.py with your actual OpenWeatherMap API key.

4.Run the application:
python weather_monitor.py

REQUIREMENTS
 1.Python 3.x
 2.requests library (install via pip install -r requirements.txt)
 
TESTING
To run tests, execute:
python -m unittest test_weather_monitor.py
